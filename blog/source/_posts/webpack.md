### webpack

##### 全局安装webpack

```javascript
npm install webpack@3.12.0 -g
```

##### 卸载webpack

```javascript
npm  uninstall  webpack  -g
```

<!--more-->

##### 打包命令webpack

```javascript
webpack ./main.js ./bundle.js
```
